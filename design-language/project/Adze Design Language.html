Adze Design Language

volcanex/adze

An editorial admin surface, dressed in warm paper and steel-blue.

Distilled from _shared/portal.html and _shared/admin.html. The system pairs italic Cardo display type with Inter UI and Monaco mono — set against a warm off-white paper with a single dusty-blue accent borrowed from the curved-axe logo.

Color paper · ink · steel

The palette reads as warm-neutral — a paper-toned bg (#f5f2ed) with desaturated dark ink and a single dusty steel-blue accent that ties back to the logo. Two accent values coexist: a deeper editorial blue for the admin surface, a softer one for the artist portal.

Voice how it talks

Quietly editorial
